

# UniCivSim

A simple simulation made to demonstrate how galactic civilizations may behave in an extremely simplified form.

## Features

- Generates a randomised star field
- Randomly generates civilizations based on the drake equation
- Civilizations expand and interact based on behaviour, desperation and technology


## Installation
*Currently only supports windows*

Download Main.exe and run it.
